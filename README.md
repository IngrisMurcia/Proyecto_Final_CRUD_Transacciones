Proyecto Final – CRUD de Transacciones

Aplicación de escritorio desarrollada en C# con Windows Forms (.NET Framework) que implementa un sistema de gestión de transacciones comerciales basado en la base de datos Northwind.

El sistema permite administrar las principales entidades del negocio mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar), incluyendo: Productos, Clientes, Empleados, Categorías, Órdenes / Transacciones

El objetivo del proyecto fue aplicar conceptos de acceso a datos, manejo de formularios, arquitectura básica por capas y manipulación de datasets tipados.

Tecnologías utilizadas: C#, .NET Framework, Windows Forms, ADO.NET, SQL Server, DataSet tipado (NorthwindDataSet), Visual Studio

Funcionalidades principales: Registro de nuevos productos, clientes y empleados; Edición de registros existentes, Eliminación de datos, Consulta y visualización en tablas, Navegación entre módulos desde un menú principal

Cómo ejecutar el proyecto: 
1. Debe Clonar el repositorio
2. Abrir la solución .sln en Visual Studio
2. Configurar la cadena de conexión en App.config
